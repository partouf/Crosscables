
Jumpknots
---------

Secure extentions for Jumpropes.
This library depends on a 3rd party library.

Cryptographic Libraries supported:
- OpenSSL
- ...


Win32 Version
-------------
- http://www.slproweb.com/products/Win32OpenSSL.html
- Install into D:\OpenSSL
- Include dir: D:\OpenSSL\include
- Lib dir: D:\OpenSSL\lib\MingW
- Libs: "D:\OpenSSL\lib\MinGW\libeay32.a" "D:\OpenSSL\lib\MinGW\ssleay32.a"
